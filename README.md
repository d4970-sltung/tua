# 歐宴泉醫師 — 以經驗承擔責任，用共識開創未來

歐宴泉醫師官方網站。本網站致力於分享歐醫師的醫療專業、願景與理念。

## 網站內容

- **關於我們 (About Section)**: 醫師簡介與經歷
- **願景 (Vision Section)**: 醫療未來的展望
- **理念 (Quote Section)**: 核心價值與格言
- **影片 (Video Section)**: 相關影音內容
- **大事記 (Timeline Section)**: 重要里程碑

## 開發指南

本專案使用 Vue 3 + Vite 開發。

### 推薦開發環境

- [VS Code](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (並禁用 Vetur)

### 安裝依賴

```bash
npm install
```

### 啟動開發伺服器

```bash
npm run dev
```

### 建置

```bash
npm run build
```
